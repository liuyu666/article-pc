本项目讲解的是一个简单的文章后台管理系统，系统的功能很简单，如下：

登录、退出；首页；内容(文章)管理：文章列表、发布文章、修改文章。

看完本篇文章你将了解学习到的知识如下：

React官方脚手架：create-react-app

react-hooks

状态管理：mobx

UI组件库：antd v5

ajax请求库：axios

路由：react-router-dom 以及 history

富文本编辑器：react-quill

CSS预编译器：sass
